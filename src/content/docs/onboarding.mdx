# Onboarding Tour
The onboarding feature is designed to guide you through parts of Lystaria that are not immediately visible when you first begin using the app. Many important tools and pages are intentionally placed inside header icons across different tabs, which keeps the interface clean but can make certain features easy to overlook. Without guidance, it is possible to use the app without ever discovering everything it offers.

This system highlights those hidden areas and shows you exactly where to tap in a step-by-step flow. It removes the need to guess or search through the interface and instead gives you a clear path to follow. Over time, this helps you build a natural understanding of where everything lives so navigation becomes intuitive.

<GradientDivider />

## How to start the onboarding tour
To begin the onboarding tour, go to the **Event tab** using the bottom navigation bar. From there, look to the **top right corner of the header**, where you will find the settings icon. This is where the onboarding entry point is located.

Once inside the Settings page, you will find the option to start the onboarding tour. Selecting it will begin guiding you through each hidden feature step by step. The process is designed to be simple and clear so you can focus on learning the structure of the app without feeling overwhelmed.
