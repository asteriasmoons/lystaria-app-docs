The Socials section connects you to my online presence. You can find links to my Threads, Instagram, and Facebook accounts, as well as my Discord server and GitHub profile.

## Navigating links
Each option includes a chevron up arrow button. Tapping this button will take you directly to the selected page or platform. This makes it easy to move from Lystaria to wherever you want to go next without needing to copy or search for links.
